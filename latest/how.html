Installation, Documentations and Tutorials¶

For installation, open the package manager mode in the Julia REPL and add MLStyle.

For more examples or tutorials, see the MLStyle-Playground project. This project will be frequently updated to demonstrate some interesting uses of MLStyle.jl.
